# Botaniq 
Um projeto web moderno e responsivo para a Botaniq, desenvolvido com foco em praticidade, organização e eficiência. A plataforma permite que clientes realizem compras de forma simples e intuitiva, enquanto oferece aos administradores ferramentas para gerenciar produtos, pedidos e informações em um único ambiente. Construído com boas práticas de desenvolvimento front-end, o sistema proporciona uma experiência fluida, escalável e otimizada para facilitar a gestão do negócio e aprimorar o atendimento aos clientes.

---
# ⚡ Funcionalidades
* Plataforma completa para consulta e gerenciamento de plantas e flores
* Sistema de autenticação e interação entre usuários
* Catálogo digital com informações detalhadas sobre cada espécie
* Guia Botânico com orientações especializadas de cultivo e manutenção
* Sistema de avaliações para compartilhamento de experiências dos clientes
* Ferramentas administrativas para controle de estoque e análise de vendas
* Relatórios gerenciais para acompanhamento das operações
* Interface responsiva, intuitiva e acessível em diferentes dispositivos

---
# Como executar
1. Faça o download ou clone o repositório da Botaniq.
2. Abra a pasta do projeto em seu editor de código preferido.
3. Localize o arquivo index.html.
4. Abra o arquivo index.html para acessar a plataforma.
