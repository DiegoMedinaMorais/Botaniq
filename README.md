# Botaniq

## 1. Nome do projeto e integrantes do grupo
- Nome do projeto: Botaniq
- Integrantes do grupo: Diego Medina Morais e Annie Vitoria Souza de Jesus

## 2. Problema & Solução
- Problema: usuários e gestores de um e-commerce de plantas precisam de uma interface centralizada para consultar produtos, cadastrar contas, enviar avaliações, consultar informações de cultivo e gerenciar o estoque e relatórios.
- Solução: o Botaniq oferece uma plataforma web responsiva com páginas de login e cadastro, catálogo de plantas, guia botânico, avaliação de clientes e áreas administrativas para controle de estoque e relatórios.
- Usuário final: clientes que desejam comprar plantas e obter informações de cultivo, além de equipe interna (estoquistas e administradores) que precisam gerenciar estoque e operações.

## 3. Escopo das Telas
- `index.html`: página inicial com banner, galeria de imagens e catálogo de produtos à venda.
- `login.html`: tela de autenticação para usuário acessar o sistema.
- `cadastro.html`: formulário de cadastro de novos usuários.
- `guiaBotanico.html`: conteúdo educativo com orientações de plantio e cards de cuidados específicos para várias espécies.
- `avaliacoes.html`: página para envio de avaliações e visualização de feedbacks de clientes.
- `relatorios.html`: área de relatórios que exibe registros de vendas ou operações para acompanhamento administrativo.
- `telaAdmin.html`: painel administrativo com navegação para relatórios e acesso a funcionalidades de gestão.
- `telaEstoquista.html`: interface de controle de estoque para aumentar ou diminuir unidades dos produtos.
- `telaPrincipal.html`: versão da página principal usada como dashboard ou acesso principal para usuários autenticados.
